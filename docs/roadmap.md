# Roadmap

This roadmap is intentionally phase-based and pragmatic.

## Next Priority Areas

1. Payroll
   - payroll calendar and pay item policy model
   - salary statement generation and approval hooks
2. Advanced Accounting
   - period close flow, trial balance, and reporting packs
   - tax/VAT workflow hardening
3. Deeper ADT/S1 Adapters
   - richer adapter contracts and mapping diagnostics
   - provider-specific parser packs (kept behind generic interfaces)
4. HWPX Export Hardening
   - structured form export quality improvements
   - template compatibility coverage for Korean enterprise documents
5. Mobile App
   - approval inbox and attendance request mobile UX
   - secure session handling for mobile clients
6. Notification Integrations
   - email/slack/kakao work alert adapters
   - escalation and retry strategy for approval reminders

## Delivery Principle

Every phase should ship usable vertical slices, not architecture-only placeholders.
