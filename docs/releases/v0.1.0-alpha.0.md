# v0.1.0-alpha.0 (Draft Prerelease Notes)

Release type: public prerelease (`alpha`)  
Status: draft notes for GitHub prerelease preparation

## Overview

`v0.1.0-alpha.0` is the first public prerelease baseline of Urimodu ERP.

This release establishes a runnable self-hosted platform with Korean workflow-oriented vertical slices across:

- auth and organization
- employee master
- files/documents/approvals/signatures/PDF
- attendance/leave/integration ingress
- expenses/finance/import-export
- ops/deployment/docs foundation

## Delivered Scope (PROMPT02-PROMPT07)

### PROMPT02 - Bootstrap/Foundation

- Monorepo and app/package skeleton
- Docker Compose stack and Helm skeleton
- Swagger bootstrap and health endpoints
- OSS governance documents and CI skeleton

### PROMPT03 - Auth/Org/Employee/Audit

- Session-based auth and RBAC
- Company/legal-entity/site/department APIs
- Employee master APIs and masking
- Mutation audit logging

### PROMPT04 - Files/Documents/Approvals

- MinIO-backed file object metadata and retrieval
- Document templates and versioning
- Approval line/step/action flow
- Signature asset registration
- HTML template rendering to PDF pipeline

### PROMPT05 - Attendance/Leave/Integrations

- Raw attendance ingestion APIs and CSV import
- Immutable raw events + normalized ledger model
- Shift policy, leave request, attendance correction starter
- Generic edge-agent ingestion path

### PROMPT06 - Expenses/Finance/Import-Export

- Expense claim starter flow
- Chart of accounts + journal entry starter
- Import/export job tracking and row-level error reporting

### PROMPT07 - Finalization and Operability

- Architecture docs and ADR hardening
- Ops docs (seed, backup/restore, production, security checklist)
- Smoke checks and docs-service smoke test coverage

## Quickstart

```bash
make bootstrap
cp .env.example .env
cp deploy/compose/.env.example deploy/compose/.env
make compose-up
pnpm --filter @korean-erp/api prisma:seed
pnpm dev
```
