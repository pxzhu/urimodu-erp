# 기능 QA: WEHAGO 레퍼런스 기반 허브 기능

## 구현 기능
- 업무 홈(`/workspace`)
  - 모듈 검색
  - 즐겨찾기 저장(로컬 스토리지)
  - 코어 KPI 카드(직원/결재/근태검토/경비 진행)
  - WEHAGO 리플릿 기반 모듈 카드(코어/협업/확장)
- 협업 허브(`/collaboration`)
  - 메신저 채널 + 대화 입력
  - 회의 일정 등록
  - 메일함 읽음/안읽음 토글
  - 드라이브 파일 검색(파일 API 연동)
  - 노트 작성
  - 게시글 등록

## 결과
- 관리자/사원 모두 허브 화면 접근 가능
- 협업 허브 탭 기능 동작 확인
- 기존 ERP 주요 화면과 병행 동작 확인
